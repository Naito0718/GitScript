# 有限群の指数 $(G:H)$

<dl><dt>

・有限群 $G$、部分群 $H$ とする。
このとき、指数：$(G;H)$
$$(G;H)=\#G/H=\#G\backslash H$$

    ・剰余類の数。

</dt><dd>

- $$\#G=(G;H)\cdot\#H\\\ \\$$

- 部分群 $H\sub G$ の元の位数は、$G$ の位数の約数。


</dd></dl>
